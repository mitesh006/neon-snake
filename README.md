# Neon Snake (React Edition)

A modern web-based version of my original terminal Snake game, rebuilt using React with responsive gameplay, swipe controls, neon visuals, and optimized real-time input handling.

This project is inspired by my earlier C++ terminal implementation and expands it into a polished browser arcade experience.

---

## 🚀 Features

- Responsive game board
- Desktop keyboard controls
- Mobile swipe controls
- Mobile pause/restart buttons
- Wrap-around gameplay
- Persistent high score storage
- Dynamic speed progression
- Neon cyber-style UI
- Smooth animations and glow effects
- Real-time optimized controls using refs

---

## 🛠 Tech Stack

- React
- JavaScript
- CSS3
- Vite

---

## 🎮 Controls

### Desktop
- WASD / Arrow Keys → Move
- SPACE → Pause / Resume
- R → Restart

### Mobile
- Swipe → Move Snake
- Pause Button
- Restart Button

---

## ⚡ Improvements Over Terminal Version

Compared to the original terminal-based Snake game, this version includes:

- Fully graphical interface
- Responsive mobile support
- Touch gesture controls
- Improved game loop handling
- Persistent high scores
- Animated neon UI
- Better performance and smoother input response
- Modern web deployment support

---

## 📱 Responsive Design

Optimized for:
